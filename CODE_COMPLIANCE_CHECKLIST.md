# Fire Alarm Code Compliance Checklist

## NFPA 72 Requirements
- [ ] Smoke detectors installed in accordance with Chapter 17
- [ ] Notification appliances meet audibility requirements
- [ ] Power supplies meet battery standby requirements
- [ ] System documentation complete

## Local Amendments
- [ ] Approved by local AHJ
- [ ] Permit applied for
- [ ] Final inspection scheduled

## Special Considerations
- [ ] High-value areas have appropriate detection
- [ ] Emergency voice/alarm communication system if required
